Tema 04 - Grundlæggende animation

På 4. tema blev vi præsenteret for javascript som bruges til at lave avancerede website og som skaber interaktive elementer, som fx. burgermenu. Vi brugtejavascript sammen med css så vi kunne lave et spil ud egen ide og koncept.

Vi fik mulighed for at afprøve vores grafiske færdigheder i Illustrator. I denne proces skulle vi også lave statemacshine- og aktivitetsdiagram, som bruges til at planlægge komplicerede interaktive flows

Vi arbejdede med .svg, .wag, .aiff og mp3. Vi skulle til slut i vores sætte sætte lyde på elementerne via Audition.

- Link til site v1 (skelet)
- Aktivitetsdiagram
