Tema 04 - Grundlæggende animation

På 4. tema blev jeg præsenteret for javascript som bruges til at lave avancerede website og som skaber interaktive elementer, som fx. burgermenu. Jeg brugte javascript sammen med css så jeg kunne lave et spil ud fra egen ide og koncept. Jeg synes dette tema var det sværeste.

Jeg fik mulighed for at afprøve mine grafiske færdigheder i Illustrator. I denne proces skulle jeg også lave statemacshine- og aktivitetsdiagram, som bruges til at planlægge komplicerede interaktive flows.

Jeg arbejdede med .svg, .wag, .aiff og mp3. Jeg skulle til slut i vores sætte sætte lyde på elementerne via Audition.

Opgaver på dette tema:
- Link til site v1 (skelet)
- Aktivitetsdiagram
- Spil
